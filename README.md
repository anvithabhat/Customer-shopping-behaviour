# 📊 Customer Behaviour Analysis (Python + SQL + Power BI)

## 📌 Project Overview

This project analyzes customer shopping behavior using Python for data cleaning, SQL for data exploration, and Power BI for interactive visualization. It provides insights into revenue trends, customer segmentation, and purchasing patterns to support data-driven decision making.

---

## 🎯 Objectives

* Clean and preprocess raw data for analysis
* Analyze customer behavior and revenue patterns
* Identify high-value customers and top-performing products
* Evaluate the impact of subscriptions and discounts
* Build an interactive dashboard for business insights

---

## 🛠️ Tools & Technologies

* Python (Pandas)
* SQL (PostgreSQL)
* Power BI
* DAX (Data Analysis Expressions)

---

## 🧹 Data Cleaning & Preprocessing (Python)

* Handled missing/null values to ensure data consistency
* Standardized column names (lowercase with underscore format)
* Renamed columns for better clarity
* Created new derived columns for analysis
* Removed unnecessary columns to optimize dataset
* Ensured correct data types for accurate processing

---

## 🔍 SQL Analysis Performed

* Revenue comparison by gender
* Identification of high-spending discounted purchases
* Top-rated products analysis
* Shipping type impact on spending
* Subscription vs non-subscription revenue comparison
* Discount usage trends across products
* Customer segmentation (New, Returning, Loyal)
* Top products by category
* Repeat customer subscription behavior
* Revenue contribution by age group
* Identification of high-demand but low-rated products

---

## 📊 Power BI Dashboard Features

* KPI Cards: Total Revenue, Avg Spending, Repeat %, Rating
* Revenue by Category
* Customer Segmentation (Donut Chart)
* Revenue Contribution by Subscription Status
* Spending Behavior by Age Group
* Top Purchased Products
* Interactive Filters (Gender, Category, Subscription, Shipping Type)

---

## 💡 Key Insights

* High repeat customer rate (~98%) indicates strong customer loyalty
* Clothing category generates the highest revenue
* Non-subscribers contribute more revenue than subscribers
* Younger customers have higher spending patterns
* Some products show high demand but lower ratings, indicating improvement areas
